Плагин для установки и настройки Личного кабинета Участника программ лояльности Loymax.

== Description ==

WordPress-плагин Loymax позволяет создать и настроить Личный кабинет (ЛК) Участника программ лояльности Loymax (ПЛ) у Вас на сайте.

# **Начало работы с плагином**
После активации плагина LoymaxWebApp запустите помощника установки и следуйте его указаниям.
Или перейдите во вкладку **Loymax** в главном меню и настройте ЛК:
1. Выберите и настройте необходимые модули во вкладке **Модули**.
2. Установите нужные значения параметров во вкладке **Настройки**.
3. Укажите нужный адрес **Loymax API** во вкладке **API** (для проверки можно указать https://demo.loymax.tech/publicapi/).
Чтобы получить ссылку на **Loymax API** своей ПЛ, обратитесь в службу технической поддержки Loymax (support@loymax.ru).
4. Установите рекомендованную тему во вкладке **Тема**.
5. Задайте название страницы, на которой будет размещен ЛК, и нажмите кнопку **"Сохранить"**.
Теперь на Вашем сайте есть страница "Личный кабинет", меню, которое актуально для навигации по разделам ЛК на этой странице, а также виджет "Loymax", с помощью которого в выбранной области сайта будет отображаться информация об Участнике ПЛ.
Вам остаётся только распорядиться страницей "Личный кабинет", одноимённым меню и виджетом "Loymax".
> Совет: можно добавить страницу "Личный кабинет" в главное меню Вашего сайта, а меню навигации по ЛК и виджет - например, в сайдбар (если вы используете тему с боковой колонкой).

Подробнее о настройке ЛК Вы можете прочитать в документации, перейдя по соответствующей ссылке на странице настроек.

== Installation ==
Загрузите плагин LoymaxWebApp на Ваш сайт, активируйте его. После чего можете переходить к созданию и настройке ЛК, предварительно ознакомившись с инструкцией в описании данного плагина.
